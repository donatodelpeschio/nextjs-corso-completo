Deployment

Vercel (Raccomandato)

  1. Push su GitHub
  2. Vai su vercel.com
  3. Click "New Project"
  4. Importa repo
  5. Deploy! ✨

Variabili d'Ambiente

# .env.local
DATABASE_URL="postgresql://..."
NEXT_PUBLIC_API_URL="https://..."

Aggiungi in Vercel: Settings → Environment Variables

Netlify

npm run build netlify deploy --prod

Docker

FROM node:20-alpine
WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
RUN npm run build
EXPOSE 3000
CMD ["npm", "start"]

Self-Hosted

npm run build npm start # Server gira su porta 3000

Static Export

// next.config.js
module.exports = {
  output: 'export',
}
npm run build # Output in /out - deploy su qualsiasi CDN
⚠️ Limitazioni Export

Con static export NON puoi usare:

  • API Routes
  • Server-Side Rendering
  • Image Optimization dinamica
  • Middleware

Performance Checklist

  • ✅ Image optimization attiva
  • ✅ Font optimization
  • ✅ ISR per contenuti dinamici
  • ✅ Analytics configurati
  • ✅ Error tracking (Sentry)
  • ✅ Sitemap e robots.txt
