Setup e Primo Progetto

Installazione

Assicurati di avere Node.js installato (versione 18.18 o superiore).

node --version # Verifica versione (serve >= 18.18)

Creare un Progetto Next.js 15+

npx create-next-app@latest my-first-nextjs-app

Durante l'installazione risponderai a queste domande:

✔ Would you like to use TypeScript? … No / Yes
✔ Would you like to use ESLint? … No / Yes
✔ Would you like to use Tailwind CSS? … No / Yes
✔ Would you like your code inside a `src/` directory? … No / Yes
✔ Would you like to use App Router? … No / Yes  ← YES!
✔ Would you like to use Turbopack for `next dev`? … No / Yes
✔ Would you like to customize the import alias? … No / Yes
💡 Configurazione Consigliata
  • TypeScript: No (per iniziare)
  • ESLint: Yes
  • Tailwind: Yes
  • src/ directory: No
  • App Router: YES! ⭐
  • Turbopack: Yes (Next.js 15+ stable)

Avviare il Server

cd my-first-nextjs-app npm run dev

Apri http://localhost:3000 - vedrai la pagina di benvenuto!

⚡ Turbopack in Next.js 15

Se hai scelto Turbopack, il dev server sarà ultra-veloce. Altrimenti puoi usarlo manualmente: npm run dev -- --turbo

Struttura del Progetto

my-first-nextjs-app/
├── app/                    ← Il cuore dell'applicazione
│   ├── layout.js          ← Layout principale
│   ├── page.js            ← Homepage (/)
│   ├── globals.css        ← Stili globali
│   └── favicon.ico
├── public/                ← File statici
├── package.json
├── next.config.js
└── .eslintrc.json
📁 La Cartella app/

Con l'App Router, tutto vive in app/. Ogni file page.js diventa una route.

La Tua Prima Pagina

// app/page.js
export default function Home() {
  return (
    <main className="p-8">
      <h1 className="text-4xl font-bold mb-4">
        La Mia Prima App Next.js 15! 🚀
      </h1>
      <p className="text-gray-300">
        Sto imparando Next.js!
      </p>
    </main>
  )
}

Salva - il browser si aggiorna automaticamente (Hot Module Replacement)!

Creare una Seconda Pagina

  1. Crea cartella app/about/
  2. Dentro crea page.js
// app/about/page.js
export default function About() {
  return (
    <main className="p-8">
      <h1 className="text-4xl font-bold">Chi Sono</h1>
      <p>Sviluppatore che impara Next.js 15!</p>
    </main>
  )
}

Vai su /about - funziona! ✨

Navigazione tra Pagine

// app/page.js
import Link from 'next/link'

export default function Home() {
  return (
    <main className="p-8">
      <h1>Homepage</h1>
      <Link 
        href="/about" 
        className="text-cyan-400 hover:underline"
      >
        Vai ad About →
      </Link>
    </main>
  )
}
⚡ Perché Link?

<Link> = navigazione client-side veloce + prefetching automatico!

🎯 Esercizio Pratico

Crea 3 pagine:

  1. Homepage (/)
  2. Projects (/projects)
  3. Contact (/contact)
  4. Menu di navigazione con Link
